Ola Bike Ride Demand Forecasting

The objective of this project was to forecast bike ride demand using historical ride booking data along with temporal, weather, and calendar-based features. The project began with data preprocessing and exploratory data analysis to identify demand patterns across different time periods.

Feature engineering was performed using variables such as hour of the day, weekday, holidays, and weather conditions. Multiple regression models were trained and compared, including Linear Regression and Random Forest. The Random Forest model demonstrated improved predictive performance and was evaluated using Mean Absolute Error (MAE). The results highlight how demand forecasting can support operational planning and resource optimization in ride-hailing services.

Technologies: Python, Pandas, Scikit-learn, Random Forest
